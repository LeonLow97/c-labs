# Hands-On C Projects

## Projects

### [Project 1: Memory Management](./project-1/)

Explore how memory is organized in C by working with the stack, heap, global variables, pointers, arrays, and structs. The project also demonstrates common memory errors—including memory leaks, use-after-free, and out-of-bounds access—and uses Valgrind to analyze them.

### [Project 2: Dynamic File Reader](./project-2/)

Build a dynamic employee file parser that reads records from a file of unknown size. The project focuses on dynamic memory allocation, buffer resizing with `realloc()`, safe parsing, error handling, and clean modular design.
